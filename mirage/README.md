## Install & Run

To make changes to the site follow the steps below:

1. Change directory to `mirage`
```shell
cd mirage
```
2. Install dependencies
```shell
npm install
```
3. Run the site in dev mode
```shell
npm run dev -- --open
```
4. Make site changes
5. Create your build directory
```shell
npm run build
```
